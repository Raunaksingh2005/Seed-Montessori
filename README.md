# 🌱 The Seed Montessori School — Official Website

Official website for **The Seed Montessori School**, an early childhood education centre located in Aerocity, Mohali, Punjab. Built with React and Firebase.

---

## Tech Stack

| Layer | Technology |
|---|---|
| Frontend | React 18, CSS Modules |
| Backend | Firebase Firestore |
| Security | Firebase Security Rules, Input Sanitization, CAPTCHA |
| Notifications | Firebase Cloud Functions + Nodemailer |
| Fonts & Icons | Google Fonts (Inter + Playfair Display), Font Awesome 6 |

---

## Features

- Fully responsive, mobile-first design
- Enrollment form with CAPTCHA — submissions saved to Firestore
- Contact form with real-time Firebase backend
- Email notifications on every new submission
- Auto-open enrollment modal on page load
- Scroll-reveal animations and image mosaic hero
- Sections: About (Vision/Mission/Philosophy), Programs, Curriculum, Daily Routine, Policies, Gallery, Contact

---

## Getting Started

```bash
git clone https://github.com/Raunaksingh2005/Seed-Montessori.git
cd Seed-Montessori
npm install
cp .env.example .env
# Add your Firebase credentials to .env
npm start
```

---

## Environment Variables

```env
REACT_APP_FIREBASE_API_KEY=
REACT_APP_FIREBASE_AUTH_DOMAIN=
REACT_APP_FIREBASE_PROJECT_ID=
REACT_APP_FIREBASE_STORAGE_BUCKET=
REACT_APP_FIREBASE_MESSAGING_SENDER_ID=
REACT_APP_FIREBASE_APP_ID=
```
